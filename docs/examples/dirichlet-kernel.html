Plot of the Dirichlet kernel used in Fouier analysis:

The unnormalized kernel is given by the analytic expression

\[ D_n (x) = \frac{ \sin \left( n + \frac{1}{2} \right) x }{ \sin \frac{1}{2} x } \]

For integral \(n\) this kernel is equal to a sum of exponential functions:

\[ D_n (x) = \sum_{ k=-n }^{ n } e^{ i k x } \]

The continuous input here allows one to see how the kernel changes in an analytic fashion, apart from whether the index is integral or not.
